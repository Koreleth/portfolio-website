## 🚀 Installation & Nutzung

1. Repository klonen:

   ```bash
   git clone https://github.com/Koreleth/Portfolio.git
   cd Portfolio
   ```
2. Node Module installieren:
    ```bash
   npm install
   ```

3. Backend starten:
    ```bash
   cd backend
   npm run dev
   ```

4. Frontend starten:
   ```bash
   cd frontend
   npm run dev
   ```

5. Gebe anschließend `localhost` im Browser ein.
